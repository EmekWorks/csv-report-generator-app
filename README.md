# CSV Report Generator App

This project is a simple Python desktop app that reads a CSV file and generates a text report.

The user selects a CSV file, and the program calculates basic statistics such as:

- number of rows
- total revenue
- average price

The final report is saved into `report.txt`.

## Features

- choose a CSV file
- read structured data from CSV
- calculate basic report values
- save report to a text file
- simple desktop GUI built with tkinter
